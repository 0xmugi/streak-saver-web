# Streak Saver Web

Browser-first landing page and dashboard concept for Streak Saver.

## Stack

- Vue 3
- Vite 5
- Tailwind CSS 4
- TypeScript

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

Current build is front-end only.
QR login and backend worker flow are presented in product UI, but server implementation is not included in this repository yet.
